# P02: Web corporativa

## Introducció

A la **tasca 8** vau triar quina seria la proposta de pàgina web que presentareu davant dels responsables de **FoodLogístics** per substituir la seva pàgina actual.

Toca, doncs, desplegar aquesta solució perquè sigui visible per al client.

## Descripció de l’activitat

Cal crear una carpeta anomenada **`docs`** a l’arrel del repositori del vostre projecte (**en local**) i, a dins, heu de copiar els arxius corresponents a la **versió definitiva** del web.

Un cop hàgiu confirmat els canvis i els hàgiu pujat a **GitHub**, configureu el desplegament a **GitHub Pages** del contingut de la carpeta **`docs`**.

Cal que **tots els membres de l’equip** facin això als seus respectius repositoris.
